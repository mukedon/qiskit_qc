Section 01

Intuitive Description

Peter Shor's 1995 code was the first demonstration that quantum error correction is possible at all — that one could protect a quantum state against both bit-flip and phase-flip errors simultaneously, without violating the no-cloning theorem. The insight is elegant: build a two-layer defense.

Core Idea
A logical qubit is encoded into 9 physical qubits by applying two nested repetition codes. The outer layer protects against phase flips ($Z$ errors) using a 3-qubit repetition code in the Hadamard basis. The inner layer protects against bit flips ($X$ errors) using three independent 3-qubit repetition codes in the computational basis.

Layer 1 — Bit-Flip Repetition Code

Recall the classical 3-bit repetition code: send 0 → 000, 1 → 111. A single bit-flip is majority-voted away. The quantum analogue works identically in the $\{|0\rangle, |1\rangle\}$ basis:

$$|0\rangle \;\xrightarrow{\text{3-bit}}\; |000\rangle, \qquad |1\rangle \;\xrightarrow{\text{3-bit}}\; |111\rangle$$

Applying an $X$ (bit-flip) to any single qubit of |000⟩ gives |100⟩, |010⟩, or |001⟩ — all distinguishable. Parity checks $Z_1Z_2$ and $Z_2Z_3$ identify the culprit without measuring the qubit itself.

Layer 2 — Phase-Flip Repetition Code

A $Z$ gate applied to any computational-basis state only imparts a global phase ($Z|0\rangle = |0\rangle$, $Z|1\rangle = -|1\rangle$), so it goes completely undetected by the $Z$-parity checks above. We must work in the Hadamard basis $\{|+\rangle, |-\rangle\}$ where $Z$ acts as a bit-flip:

$$Z|{+}\rangle = |-\rangle, \qquad Z|{-}\rangle = |{+}\rangle$$

So we apply another 3-qubit repetition — but now across the $|{+}\rangle / |{-}\rangle$ states of three "super-qubits":

$$|{+}\rangle \;\xrightarrow{\text{3-phase}}\; |{+}\rangle|{+}\rangle|{+}\rangle, \qquad |{-}\rangle \;\xrightarrow{\text{3-phase}}\; |{-}\rangle|{-}\rangle|{-}\rangle$$

A $Z$ flip on any of the three blocks turns one $|{+}\rangle$ into $|{-}\rangle$. An $X$-type parity check across blocks detects this.

Concatenating Both Layers

The full Shor encoding first applies the phase-flip repetition, spreading the logical qubit across three "blocks," then applies the bit-flip repetition within each block, expanding each block from 1 to 3 qubits. The result is 9 physical qubits encoding 1 logical qubit.

Logical ─────────────────────────────── Outer (phase) repetition ──── Inner (bit) repetition ──► |ψ⟩ ─── Phase-flip code ───► |+/−⟩ ⊗ |+/−⟩ ⊗ |+/−⟩ ─── Bit-flip code ───► 9 qubits Block 1: qubits q₀ q₁ q₂ ← 3-qubit bit-flip code on first |+/−⟩ Block 2: qubits q₃ q₄ q₅ ← 3-qubit bit-flip code on second |+/−⟩ Block 3: qubits q₆ q₇ q₈ ← 3-qubit bit-flip code on third |+/−⟩
Why does this correct arbitrary errors?
Any single-qubit error $E$ on qubit $i$ can be decomposed into Pauli operators: $E = a\,I + b\,X + c\,Y + d\,Z$. Because the syndrome measurement projects the error onto a specific Pauli ($X$, $Z$, $XZ$, or $I$), it suffices to correct those four cases. The Shor code handles $X$ and $Z$ independently — and since $Y = iXZ$, it handles $Y$ as well.
Section 02

Mathematical Formulation

Logical Codewords

Define the following shorthand for the two "inner" states of each 3-qubit block:

$$|\overline{0}_B\rangle \;=\; \frac{|000\rangle + |111\rangle}{\sqrt{2}}, \qquad |\overline{1}_B\rangle \;=\; \frac{|000\rangle - |111\rangle}{\sqrt{2}}$$

Note that $|\overline{0}_B\rangle = H^{\otimes 3}|000\rangle$ and $|\overline{1}_B\rangle = H^{\otimes 3}|111\rangle$ (up to normalisation), which will be useful when constructing the circuit. The full logical codewords of the Shor code are then:

$$\boxed{|0_L\rangle \;=\; \frac{1}{2\sqrt{2}}\bigl(|000\rangle+|111\rangle\bigr)\bigl(|000\rangle+|111\rangle\bigr)\bigl(|000\rangle+|111\rangle\bigr)}$$
$$\boxed{|1_L\rangle \;=\; \frac{1}{2\sqrt{2}}\bigl(|000\rangle-|111\rangle\bigr)\bigl(|000\rangle-|111\rangle\bigr)\bigl(|000\rangle-|111\rangle\bigr)}$$

An arbitrary logical qubit $|\psi_L\rangle = \alpha|0_L\rangle + \beta|1_L\rangle$ encodes $|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$. The $\pm$ signs separating the three tensor-product blocks carry the phase-flip information; the repetition within each block carries the bit-flip information.

Encoding Map

Let us trace $|0\rangle$ through the encoding procedure step by step.

  1. Start with $|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$ on qubit $q_0$; ancilla qubits $q_1\ldots q_8$ are in $|0\rangle$.
  2. Apply $\mathrm{CNOT}_{q_0 \to q_3}$ and $\mathrm{CNOT}_{q_0 \to q_6}$ (the outer phase-flip repetition): $$\alpha|000\rangle + \beta|111\rangle \quad \text{(on } q_0, q_3, q_6\text{)}$$
  3. Apply $H_{q_0}$, $H_{q_3}$, $H_{q_6}$: $$\alpha|{+}{+}{+}\rangle + \beta|{-}{-}{-}\rangle$$ $$= \frac{\alpha}{\sqrt{8}}\bigl(|000\rangle+|111\rangle\bigr)^{\otimes3} + \frac{\beta}{\sqrt{8}}\bigl(|000\rangle-|111\rangle\bigr)^{\otimes3}$$
  4. For each block $k \in \{0,3,6\}$: apply $\mathrm{CNOT}_{q_k \to q_{k+1}}$ and $\mathrm{CNOT}_{q_k \to q_{k+2}}$ (inner bit-flip repetition). The encoding is now complete.

Distance and Code Parameters

The Shor code is a $[[9, 1, 3]]$ code: it uses $n = 9$ physical qubits, encodes $k = 1$ logical qubit, and has distance $d = 3$, meaning it can correct any single-qubit error ($t = \lfloor(d-1)/2\rfloor = 1$) and detect any two-qubit error.

Hamming Bound (Quantum Singleton)
The quantum Hamming bound states that for an $[[n, k, d]]$ code correcting $t$ errors: $$\sum_{j=0}^{t} \binom{n}{j} 3^j \;\leq\; 2^{n-k}$$ For $[[9,1,3]]$ with $t=1$: $\binom{9}{0}3^0 + \binom{9}{1}3^1 = 1 + 27 = 28 \leq 2^{9-1} = 256$. ✓ (The code is far from tight — later codes like Steane's $[[7,1,3]]$ are more efficient.)
Section 03

Stabilizer Structure

The Shor code is a stabilizer code. Its codespace is the simultaneous $+1$ eigenspace of 8 independent Pauli operators called stabilizer generators, forming the stabilizer group $\mathcal{S}$.

The 8 Generators

Generator Operator (qubits 1–9) Detects Type
$g_1$ Z₁Z₂ on qubits 1,2 $X$ error on $q_1$ or $q_2$ in block 1 ZZ
$g_2$ Z₂Z₃ on qubits 2,3 $X$ error on $q_2$ or $q_3$ in block 1 ZZ
$g_3$ Z₄Z₅ on qubits 4,5 $X$ error on $q_4$ or $q_5$ in block 2 ZZ
$g_4$ Z₅Z₆ on qubits 5,6 $X$ error on $q_5$ or $q_6$ in block 2 ZZ
$g_5$ Z₇Z₈ on qubits 7,8 $X$ error on $q_7$ or $q_8$ in block 3 ZZ
$g_6$ Z₈Z₉ on qubits 8,9 $X$ error on $q_8$ or $q_9$ in block 3 ZZ
$g_7$ X₁X₂X₃X₄X₅X₆ $Z$ error in block 1 or block 2 XXXXXX
$g_8$ X₄X₅X₆X₇X₈X₉ $Z$ error in block 2 or block 3 XXXXXX

The first six generators ($g_1$–$g_6$) are $Z$-type checks: they detect bit-flip ($X$) errors within each 3-qubit block. The last two ($g_7$, $g_8$) are $X$-type checks: they detect phase-flip ($Z$) errors between pairs of blocks. Together they span the 8-dimensional stabilizer group, leaving a 1-qubit logical subspace.

Logical Operators

OperatorExpressionAction
$\bar{Z}_L$ Z₁Z₄Z₇ (one $Z$ per block) $|0_L\rangle \mapsto |0_L\rangle$, $|1_L\rangle \mapsto -|1_L\rangle$
$\bar{X}_L$ X₁X₂X₃X₄X₅X₆X₇X₈X₉ $|0_L\rangle \mapsto |1_L\rangle$, $|1_L\rangle \mapsto |0_L\rangle$

Both operators commute with all stabilizer generators, but anticommute with each other ($\bar{X}_L\bar{Z}_L = -\bar{Z}_L\bar{X}_L$), as required for a qubit algebra.

Syndrome Decoding Table

Measuring the 8 stabilizer generators yields an 8-bit syndrome $\mathbf{s} \in \{0,1\}^8$. A $+1$ measurement gives bit $0$; a $-1$ measurement gives bit $1$. The syndrome uniquely identifies the error (up to stabilizer equivalence):

Bit-Flip ($X$) Errors

An $X$ on qubit $q_i$ flips the parity checks of the two $ZZ$ generators touching $q_i$ within its block. The block is identified first; then the qubit within the block.

e.g. X on q₂: s = [1,1,0,0,0,0,0,0]
→ block 1, middle qubit → apply X₂
Phase-Flip ($Z$) Errors

A $Z$ on any qubit in block $b$ is equivalent to $\bar{Z}_b$ — a logical phase flip on that block. The $XX$ generators between blocks reveal which block is odd.

e.g. Z on q₅: s = [0,0,0,0,0,0,1,1]
→ block 2 flipped → apply Z₄ (or any in block 2)
Combined ($Y = iXZ$) Errors

A $Y$ error triggers both a $ZZ$ syndrome (from the $X$ component) and an $XX$ syndrome (from the $Z$ component). Apply both corrections independently.

e.g. Y on q₃: s = [0,1,0,0,0,0,1,0]
→ X correction on q₃, Z correction on block 1

1. Encoding Circuit

┌───┐ d[0] ──■────■──┤ H ├──■────■── │ │ └───┘┌─┴─┐ │ d[1] ──┼────┼───────┤ X ├──┼── │ │ └───┘┌─┴─┐ d[2] ──┼────┼────────────┤ X ├ ┌─┴─┐ │ ┌───┐ └───┘ d[3] ┤ X ├──┼──┤ H ├──■────■── └───┘ │ └───┘┌─┴─┐ │ d[4] ───────┼───────┤ X ├──┼── │ └───┘┌─┴─┐ d[5] ───────┼────────────┤ X ├ ┌─┴─┐┌───┐ └───┘ d[6] ─────┤ X ├┤ H ├──■────■── └───┘└───┘┌─┴─┐ │ d[7] ───────────────┤ X ├──┼── └───┘┌─┴─┐ d[8] ────────────────────┤ X ├ └───┘
Python · Qiskit
def shor_encode(qc, d):
    """
    Encode d[0] (logical input) into 9 data qubits d[0]..d[8].
    Assumes d[1]..d[8] are initialised to |0⟩.
    """
    # ── Outer (phase-flip) repetition ──────────────────────────
    # Copy d[0] to d[3] and d[6] in the computational basis
    qc.cx(d[0], d[3])
    qc.cx(d[0], d[6])

    # Rotate all three "block heads" into the X-basis
    qc.h(d[0]);  qc.h(d[3]);  qc.h(d[6])

    # ── Inner (bit-flip) repetition — one per block ────────────
    for base in [0, 3, 6]:
        qc.cx(d[base], d[base+1])
        qc.cx(d[base], d[base+2])

# ── Build and draw the encoding circuit ─────────────────────────
d = QuantumRegister(9, 'd')
qc_shor_enc = QuantumCircuit(d)
shor_encode(qc_shor_enc, d)
qc_shor_enc.draw('mpl')

2. Syndrome Measurement

Each ancilla qubit measures one stabilizer generator non-destructively via the standard ancilla-mediated stabilizer circuit: Hadamard the ancilla, apply controlled Paulis from ancilla onto data qubits, Hadamard again, then measure.

Python · Qiskit
def measure_stabilizer_Z(qc, anc, qubits, syn_bit, s):
    """Measure a ZZ...Z stabilizer. Ancilla encodes parity of qubits."""
    for q in qubits:
        qc.cx(q, anc)       # CX measures Z parity (ancilla in |0⟩)
    qc.measure(anc, s[syn_bit])
    qc.reset(anc)             # Reset ancilla for reuse

def measure_stabilizer_X(qc, anc, qubits, syn_bit, s):
    """Measure an XX...X stabilizer. Ancilla encodes X-basis parity."""
    qc.h(anc)                 # Ancilla into |+⟩
    for q in qubits:
        qc.cx(anc, q)        # CX with ancilla as control
    qc.h(anc)                 # Rotate back
    qc.measure(anc, s[syn_bit])
    qc.reset(anc)

def shor_measure_syndrome(qc, d, a, s):
    """
    Measure all 8 stabilizer generators.
    s[0..5] ← ZZ checks (bit-flip detectors)
    s[6..7] ← XX checks (phase-flip detectors)
    """
    # ZZ stabilizers: g1..g6
    measure_stabilizer_Z(qc, a[0], [d[0], d[1]], 0, s)  # Z₁Z₂
    measure_stabilizer_Z(qc, a[1], [d[1], d[2]], 1, s)  # Z₂Z₃
    measure_stabilizer_Z(qc, a[2], [d[3], d[4]], 2, s)  # Z₄Z₅
    measure_stabilizer_Z(qc, a[3], [d[4], d[5]], 3, s)  # Z₅Z₆
    measure_stabilizer_Z(qc, a[4], [d[6], d[7]], 4, s)  # Z₇Z₈
    measure_stabilizer_Z(qc, a[5], [d[7], d[8]], 5, s)  # Z₈Z₉
    # XX stabilizers: g7, g8
    measure_stabilizer_X(qc, a[6], d[0:6], 6, s)  # X₁..X₆
    measure_stabilizer_X(qc, a[7], d[3:9], 7, s)  # X₄..X₉

3. Classical Decoder

Python · Qiskit
def shor_decode_syndrome(syndrome_bits):
    """
    Given an 8-bit syndrome list [s0..s7], return (qubit_index, error_type).
    qubit_index: 0-8 (data qubit), or None for no error.
    error_type: 'X', 'Z', 'Y', or None.
    """
    s = syndrome_bits

    # ── Bit-flip (X) detection from ZZ checks ──────────────────
    # Each block: [s0,s1], [s2,s3], [s4,s5] → which qubit in block
    flip_lookup = {
        (0,0): None,   # no bit-flip in this block
        (1,0): 0,      # q0 / q3 / q6 (first in block)
        (1,1): 1,      # q1 / q4 / q7 (middle)
        (0,1): 2,      # q2 / q5 / q8 (last)
    }
    x_errors = []
    for blk, base in enumerate([0, 2, 4]):
        key = (s[base], s[base+1])
        offset = flip_lookup[key]
        if offset is not None:
            x_errors.append(blk * 3 + offset)

    # ── Phase-flip (Z) detection from XX checks ─────────────────
    # s[6], s[7] identify which block has a phase flip
    phase_lookup = {
        (0,0): None,   # no phase flip
        (1,0): 0,      # block 1 (apply Z to any qubit in block 1)
        (1,1): 1,      # block 2
        (0,1): 2,      # block 3
    }
    z_block = phase_lookup[(s[6], s[7])]
    z_errors = [z_block * 3] if z_block is not None else []

    return x_errors, z_errors  # apply X gates at x_errors, Z gates at z_errors

4. Full Round-Trip: Encode → Error → Correct

Python · Qiskit
def shor_full_circuit(error_qubit=None, error_type='X', initial_state='0'):
    """
    Build and run a complete Shor error correction round.

    Parameters
    ----------
    error_qubit  : int or None  – which data qubit to corrupt (0-8), None = no error
    error_type   : 'X' | 'Z' | 'Y' – Pauli error to inject
    initial_state: '0' | '1' | '+' | '-'
    """
    # ── Build circuit ──────────────────────────────────────────
    d = QuantumRegister(9,  'd')
    a = QuantumRegister(8,  'a')
    s = ClassicalRegister(8, 's')
    qc = QuantumCircuit(d, a, s)

    # Prepare logical input state
    if initial_state == '1':  qc.x(d[0])
    if initial_state == '+':  qc.h(d[0])
    if initial_state == '-':  qc.x(d[0]); qc.h(d[0])
    qc.barrier()

    # Stage 1: Encode
    shor_encode(qc, d)
    qc.barrier()

    # Stage 2: Inject error (simulates decoherence)
    if error_qubit is not None:
        if 'X' in error_type: qc.x(d[error_qubit])
        if 'Z' in error_type: qc.z(d[error_qubit])
    qc.barrier()

    # Stage 3: Measure syndrome
    shor_measure_syndrome(qc, d, a, s)

    # ── Simulate ───────────────────────────────────────────────
    sim = AerSimulator()
    result = sim.run(qc, shots=1).result()
    syndrome_str = list(result.get_counts().keys())[0]
    syn_bits = [int(b) for b in reversed(syndrome_str)]

    # ── Decode and report ───────────────────────────────────────
    x_corr, z_corr = shor_decode_syndrome(syn_bits)
    print(f"Syndrome : {syndrome_str}")
    print(f"X corrections needed at qubits: {x_corr}")
    print(f"Z corrections needed at qubits: {z_corr}")
    return qc, syn_bits, x_corr, z_corr
Section 05

Error Correction Examples

We walk through three canonical scenarios — a bit-flip, a phase-flip, and a combined $Y$ error — showing exactly which syndrome bits flip and how the decoder responds.

Example 1 — Bit-Flip ($X$) on qubit $q_4$

Suppose the logical state is $|0_L\rangle$ and qubit $q_4$ (the middle qubit of block 2) undergoes a bit-flip. The encoded state shifts from $|\overline{0}_B\rangle|\overline{0}_B\rangle|\overline{0}_B\rangle$ to $|\overline{0}_B\rangle \cdot X_4|\overline{0}_B\rangle \cdot |\overline{0}_B\rangle$.

Syndrome Analysis

$Z_3Z_4$ (generator $g_3$) anticommutes with $X_4$: flips to $-1$ → s[2] = 1
$Z_4Z_5$ (generator $g_4$) anticommutes with $X_4$: flips to $-1$ → s[3] = 1
All $X$-type generators commute with $X_4$ (same Pauli type) → s[6] = s[7] = 0

Syndrome: s = [0, 0, 1, 1, 0, 0, 0, 0]
Interpretation: Both $ZZ$ checks of block 2 fire → qubit $q_4$ (middle of block 2).
Correction: Apply $X_4$. No $Z$ correction needed.

Python · Qiskit — Example 1
# Example 1: X error on qubit 4 (middle of block 2)
qc, syn, x_c, z_c = shor_full_circuit(
    error_qubit=4,
    error_type='X',
    initial_state='0'
)

# Expected output:
# Syndrome : 00110000   (bits s2, s3 = 1)
# X corrections needed at qubits: [4]
# Z corrections needed at qubits: []

# Apply correction (in a real device, done via classical feed-forward)
for q in x_c: qc.x(d[q])
for q in z_c: qc.z(d[q])

Example 2 — Phase-Flip ($Z$) on qubit $q_7$

Now suppose qubit $q_7$ (first qubit of block 3) suffers a phase flip $Z_7$. Within block 3, the state is $|\overline{0}_B\rangle = (|000\rangle + |111\rangle)/\sqrt{2}$. Applying $Z_7$ gives $(|000\rangle - |111\rangle)/\sqrt{2} = |\overline{1}_B\rangle$ — the block has been flipped in the Hadamard basis.

Syndrome Analysis

$Z$ operators commute with all other $Z$ operators → s[0]..s[5] = 0
$X_1\ldots X_6$ (generator $g_7$): acts on block 1 and 2 only, does not touch block 3 → s[6] = 0
$X_4\ldots X_9$ (generator $g_8$): anticommutes with $Z_7$ → s[7] = 1

Syndrome: s = [0, 0, 0, 0, 0, 0, 0, 1]
Interpretation: Only $g_8$ fires → block 3 has a phase flip.
Correction: Apply $Z_6$ (or any qubit in block 3). No $X$ correction needed.

Python · Qiskit — Example 2
# Example 2: Z error on qubit 7 (first qubit of block 3)
# Try with a superposition input |+⟩ to stress-test phase correction

    error_qubit=7,
    error_type='Z',

# X corrections needed at qubits: []

# The logical |+⟩ state is recovered correctly — phase information preserved

Example 3 — Combined ($Y$) Error on qubit $q_2$

A $Y = iXZ$ error on qubit $q_2$ (last qubit of block 1) acts as both a bit-flip and a phase-flip simultaneously. The two error components are independent and are caught by different parts of the syndrome.

Syndrome Analysis

From the $X$ component of $Y$:
$Z_2Z_3$ anticommutes with $X_2$ → s[1] = 1; $Z_1Z_2$ also anticommutes → s[0] = 1 (wait — let's be careful)
$g_1 = Z_1Z_2$: $Z_2$ anticommutes with $X_2$ but $Z_1$ commutes → net anticommute → s[0] = 0
$g_2 = Z_2Z_3$: $Z_2$ anticommutes with $X_2$; $Z_3$ commutes → net anticommute → s[1] = 1

From the $Z$ component of $Y$:
$g_7 = X_1\ldots X_6$: anticommutes with $Z_2$ → s[6] = 1
$g_8 = X_4\ldots X_9$: commutes with $Z_2$ (acts on different qubits) → s[7] = 0

Syndrome: s = [0, 1, 0, 0, 0, 0, 1, 0]
Interpretation: Only $g_2$ fires (ZZ) → last qubit of block 1 has a bit-flip. Only $g_7$ fires (XX) → block 1 has a phase-flip.
Correction: Apply $X_2$ then $Z_0$ (or any qubit in block 1). Logical state fully restored.

Python · Qiskit — Example 3
# Example 3: Y error on qubit 2 (last qubit of block 1)
qc, syn, x_c, z_c = shor_full_circuit(
    error_qubit=2,
    error_type='Y',   # = 'XZ' in our convention
    initial_state='1'

# Syndrome : 01000010   (bits s1=1, s6=1)
# X corrections needed at qubits: [2]   ← undo bit-flip
# Z corrections needed at qubits: [0]   ← undo phase-flip on block 1

✦
Summary — What the Shor Code Achieves

The Shor code demonstrates that all single-qubit errors are correctable using only local measurements and classical feedback. Its two-layer structure maps onto exactly the two independent Pauli directions: $Z$-type checks catch $X$ errors in space (within blocks), and $X$-type checks catch $Z$ errors across blocks. While it is not the most efficient code ($[[9,1,3]]$ vs the optimal $[[5,1,3]]$), its architectural clarity makes it the ideal pedagogical entry point into CSS codes and the stabilizer formalism. The Steane code — next in this series — achieves the same distance with only 7 qubits by exploiting classical Hamming codes more efficiently.
